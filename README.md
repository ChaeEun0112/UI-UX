# 🍞채식빵_ZIP -  비건브레드 레시피 모음집
> 앱 서비스 개발 기간 : 2023.12

![Intro](
## 💬의미
>- 비건브레드 레시피를 제공하는 모바일 애플리케이션
>- 건강하고 맛있는 비건브레드를 쉽게 만들 수 있도록 돕는 앱 서비스
## 💡개요
- 앱 이름 '채식빵_ZIP
  - 'Vegan Bread'의 한글 번역과 'ZIP'을 결합하여, 비건빵의 모음집 의미
  - 발음상 '채식빵집'처럼 들리도록 하여 친근한 이미지 강조
- 로고 디자인
  - 식빵 모양과 비건을 상징하는 새싹 모양을 결합하여 비건빵을 시각적으로 표현
- 글씨 디자인
  - 글씨 부분은 검정색과 흰색을 사용하여 깔끔한 가독성을 제공
  - 전체적인 색감은 빵과 잘 어울리는 연한 베이지 계열부터 진한 고동색 계열까지 사용
∴ 동일 색상 내에서 차이를 주는 '톤 온 톤' 느낌을 활용
- 디자인 요소
  - 테두리 없이 채우기 기능을 사용
  - 같은 갈색이지만 투명도에 따라 차이점을 둠
## 🛠️제작 과정
- Adobe XD을 활용하여 앱의 UI/UX 디자인 구현
- 색상 선택에 있어 빵을 연상시키는 갈색 계열을 사용하여 시각적 일관성 유지
- 사용자 편의를 고려하여 다양한 카테고리와 메뉴 구성
- 직관적인 네비게이션을 통해 쉽게 탐색 가능
- 각 레시피의 정보와 필요 도구 및 재료를 한눈에 볼 수 있는 구성으로 사용자 경험 개선
## 💻주요 기능
- Login
 - 인트로 다음에 나오는 페이지
 - 앱의 아이콘과 Welcome이라는 글씨 삽입
 - 하단에는 다양한 방법으로 로그인 할 수 있도록 구성
- Main(총 3개로 구성)
  - 하단에 홈 버튼을 눌렀을 때 상단에 있는 사진 ㅂ분에 동그라미 아이콘을 눌러 화면이 넘어가는 느낌 표현
  - 레시피를 가장 많이 보는 Best5 선정
  - 그림을 눌렀을 때 바로 레시피 화면으로 넘어갈 수 있도록 구성
  - 빵을 만들 때 쓰이는 도구와 재료를 따로 메뉴창이나 아래 있는 스토어 아이콘을 누르지 앟아도 바로 볼 수 있도록 구성
- Menu
  - 큰 카테고리 선택 시 그 화면으로 넘어가게 구성
  - 큰 카테고리 밑에 작은 카테고리 나열
  - 작은 카테ㅗ리 선택 시 그 빵에 맞는 레시피 화면으로 구성
- Search
  - 상단에 있는 돋보기 아이콘 클릭 시 검색으로 찾을 수 있도록 구성
- Favorite
  - 자신이 좋아하거나 다시 보고 싶은 레시피가 있다면
