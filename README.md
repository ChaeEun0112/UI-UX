# 🍞채식빵_ZIP -  비건브레드 레시피 모음집
> 앱 서비스 개발 기간 : 2023.12

## 💬의미
>- 비건브레드 레시피를 제공하는 모바일 애플리케이션
>- 건강하고 맛있는 비건브레드를 쉽게 만들 수 있도록 돕는 앱 서비스
## 💡개요
- 앱 이름 '채식빵_ZIP
  - 'Vegan Bread'의 한글 번역과 'ZIP'을 결합하여, 비건빵의 모음집 의미
  - 발음상 '채식빵집'처럼 들리도록 하여 친근한 이미지 강조
- 로고 디자인
  - 식빵 모양과 비건을 상징하는 새싹 모양을 결합하여 비건빵을 시각적으로 표현
- 글씨 디자인
  - 글씨 부분은 검정색과 흰색을 사용하여 깔끔한 가독성을 제공
  - 전체적인 색감은 빵과 잘 어울리는 연한 베이지 계열부터 진한 고동색 계열까지 사용
∴ 동일 색상 내에서 차이를 주는 '톤 온 톤' 느낌을 활용
- 디자인 요소
  - 테두리 없이 채우기 기능을 사용
  - 같은 갈색이지만 투명도에 따라 차이점을 둠
## 🛠️제작 과정
- Adobe XD을 활용하여 앱의 UI/UX 디자인 구현
- 색상 선택에 있어 빵을 연상시키는 갈색 계열을 사용하여 시각적 일관성 유지
- 사용자 편의를 고려하여 다양한 카테고리와 메뉴 구성
- 직관적인 네비게이션을 통해 쉽게 탐색 가능
- 각 레시피의 정보와 필요 도구 및 재료를 한눈에 볼 수 있는 구성으로 사용자 경험 개선
