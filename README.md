# 🍞채식빵_ZIP -  비건브레드 레시피 모음집
> 앱 서비스 개발 기간 : 2023.12

![Intro](
## 💬의미
- 비건브레드 레시피를 제공하는 모바일 애플리케이션
- 건강하고 맛있는 비건브레드를 쉽게 만들 수 있도록 돕는 앱 서비스
## 💡개요
- 앱 이름 '채식빵_ZIP
  - 'Vegan Bread'의 한글 번역과 'ZIP'을 결합하여 비건빵의 모음집 의미
  - 발음상 '채식빵집'처럼 들리도록 하여 친근한 이미지 강조
- 로고 디자인
  - 식빵 모양과 비건을 상징하는 새싹 모양을 결합하여 비건빵을 시각적으로 표현
- 글씨 디자인
  - 검정색과 흰색을 사용하여 깔끔한 가독성을 제공
  - 빵과 어울리는 베이지 ~ 고동색 계열 색상 사용
∴ 동일 색상 내에서 차이를 주는 'Tone on Tone' 디자인 활용
- 디자인 요소
  - 테두리 없이 채우기 기능 사용
  - 같은 갈색 계열에서 투명도 차이로 디자인 표현
## 🛠️제작 과정
- Adobe XD을 활용하여 앱 UI/UX 디자인 구현
- 빵을 연상시키는 갈색 계열 사용으로 시각적 일관성 유지
- 사용자 편의를 고려한 카테고리 및 메뉴 구조 설계
- 직관적인 네비게이션으로 쉽게 탐색 가능하도록 구성
- 레시피,도구,재료 정보를 한눈에 확인할 수 있는 UI 설계
## 💻주요 기능
- Login
  - Intro 이후 나타나는 로그인 화면
  - 앱 아이콘과 "Welcome" 메시지, 다양한 로그인 방식 제공
- Main
  - Home 버튼으로 화면 전환 UI
  - Best 5 레시피 제공 및 이미지 클릭 시 레시피 화면 이동
  - 베이킹 도구와 재료 바로 확인 가능
- Menu
  - 큰 카테고리 선택 후 세부 카테고리 탐색
  - 선택한 빵의 레시피 화면으로 이동
- Search
  - 돋보기 아이콘으로 레시피 검색 가능
- Favorite
  - 좋아하거나 다시 보고 싶은 레시피 저장
- My page
  - 프로필, 아이디, Favorite 수, 주문 정보 확인 
  - Log Out 클릭 시 로그인 화면으로 이동
- Bread Category
  - 기본 빵 / 스타일리시한 빵 / 리치한 빵 카테고리 제공
  - 빵 선택 시 해당 레피 화면으로 이동
  - Favorite 추가 및 레시피 공유 가능
- Food Category
  - 빵과 잘 어울리는 음식 카테고리 제공
  - Category 1 → Category 2로 이동 가능
  - 이미지 클릭 시 레시피 화면 표시
- Recipe
  - Recipe 동영상 제공 (시작 / 완료 사진으로 표현)
  - 버튼 및 슬라이드를 통한 화면 전환 가능
  - 빵 이름, 설명, 재료, 과정 확인 가능
- Store
  - 베이킹 도구 및 재료 구매 가능
  - 상품 이름과 가격 표시
  -  Favorite 추가 및 주문 가능
